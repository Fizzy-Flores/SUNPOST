# Sunpost Website

## Quick Start

### Local
```bash
cd frontend
npm run dev:full
```

Open: `http://localhost:5173`

### Tailscale
Use Tailscale on your device and access:
- `http://100.75.71.126:5173`
- Backend: `http://100.75.71.126:8000`

## Recent updates

- Backend dashboard available at `http://<host>:8000/`
